# Plan Travel

## Objetivo

Crear una app que facilita la toma de decisiones en un viaje.

## Público objetivo

Viajeros que quieren planear un viaje sin información concisa u organizada. (Solo dentro de Argentina por ahora.)

## Funcionalidades principales

- Creación de grupos de viaje e invitaciones.
- Selección de destinos y actividades.
- Recomendación de destinos y actividades.
- División de gastos.
- Gestor de documentos importantes.
- Avisos/tips para el viaje en ese destino.

## Alcance actual

El contenido y las recomendaciones están enfocadas en Argentina.

## Requisitos

- Android Studio (última versión estable).
- JDK 11.
